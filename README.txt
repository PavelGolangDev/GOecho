Программа написана на языке Go, в среде разработки Visual Studio Code. 
Для того что бы ее запустить, требуется установленный Go.
Программа запускается помощью командной строки с правами администратора. Если Go введен в системную переменную PATH,то требуется написать следующее "go build GOLANG.go".
Если в переменную PATH ничего не добавляли, тогда C:\(Расположение go)\go.exe(Расположение go) run C:\(Расположение файла)\GOLANG.go
	В данной программе используется фреймворк echo. Для того что бы его подключить импортируются пакеты "net/http" и "github.com/labstack/echo". Если
echo не скачивается автоматически, надо зайти в командную строку и прописать go get "github.com/labstack/echo".
Что бы воспользоваться программой, надо установить расширение Postman или Restlet клиент в браузере Google Chrome.
Так же можно скачать приложение Postman с официального сайта.
	Для того что бы запустить программу, надо вставить ссылку на свой localhost(который указан в программе) в специальную строку в приложении Postman.
Чтоб начать работу после URL надо добавить /patients.
	Для того что бы добавить нового пациента в базу надо выбрать из выпадающего списка POST, перейти в вкладку body,
поставить флажек на raw и выбрать JSON(application/json). Далее в пустое поле вводим { "name":"Фамилия","diag":"Диагноз","vrach":"Фамилия врача" }
После нажатии кнопки SEND в самом нижнем поле будет результат в виде{"id": 1,"name": "Фамилия","diag": "Диагноз","vrach": "Фамилия врача"}
С каждым новым пациентом, id увеличивается на 1.
Для того что бы обновить информацию, выбираем в списке PUT, в URL после своего URL надо вставить нужное ID, и писать информацию в той же форме.
Что бы посмотреть информацию про пациента надо ввести после URL его id. Что бы удалить надо сделать тоже самое.
	Добавлены два вида авторизации: Basic Auth и Cookie login. 
Для того что-бы авторизоваться через Cookie, в URL надо вписать:http://localhost:1323/login?username=вашЛогин&password=ВашПароль
В случае введения неправильного пароля вы увидите сообщение:You entered the wrong password!
Если же пароль верен, сообщение будет следующим:You are authorized!
Что бы авторизоваться через Basic Auth надо в приложении Postman ввести URL: http://localhost:1323/admin/main и в вкладке Auhtorization выбрать Basic Auth.
В соответсвующих полях ввести пароль и логин, нажать Send. В случае успеха вы увидите надпись:admin main page.
